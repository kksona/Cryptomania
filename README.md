# Cryptomania 

A sleek, responsive cryptocurrency tracking dashboard built with React. This application provides real-time insights into market trends, price fluctuations, and detailed coin statistics.

## Live Deployment
**Check out the live app here:** [https://cryptomania-roan.vercel.app/](https://cryptomania-roan.vercel.app/)

---

## Features
* **Live Market Data:** Real-time tracking of top cryptocurrencies.
* **Interactive Visualizations:** Dynamic charts to monitor price history.
* **Deep Insights:** Comprehensive data including market cap, volume, and supply.
* **Responsive UI:** Fully optimized for mobile, tablet, and desktop screens.

## Tech Stack
* **Frontend:** React.js
* **State Management:** (e.g., Hooks)
* **Styling:** (CSS Modules / Tailwind)
* **Deployment:** Vercel
